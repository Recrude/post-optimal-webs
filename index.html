POST-OPTIMAL WEBS ARCHIVE

WWW.SPECULATIVEWEBS.VERCEL.APP

This project aims to form a forum for discourse by collecting websites that ask fundamental questions about future society, current technology culture, and human values beyond providing information using an open and experimental medium called a website. This archive pre-experiments the possibilities, risks, and the beginning of a new order in which speculative design has not yet arrived by collecting various web with speculative attributes such as a web that asks speculative questions, a web that contains speculative design, and a web that unfamiliarizes existing conventional wisdom. Recording and sharing speculative attributes in a space where imagination and critical discourse intersect in real time will be an act of opening a forum for collective imagination and dialogue about the future we will create.
